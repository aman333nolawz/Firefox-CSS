# Firefox-CSS
My Firefox userChrome.css based on cascade. This is just a fork of https://github.com/andreasgrafen/cascade

# Installation
1. Type `about:config` into your URL bar. Click on the I accept the risk button if you're shown a warning.
2. Seach for `toolkit.legacyUserProfileCustomizations.stylesheets` and set it to true
3. Go to your profile folder:
  - Linux: `$HOME/.mozilla/firefox/######.default-release/`
  - MacOS: `Users/[USERNAME]/Library/Application Support/Firefox/Profiles/######.default-release`
  - Windows: `C:\Users\[USERNAME]\AppData\Roaming\Mozilla\Firefox\Profiles\######.default-release`
4. Copy the `chrome` folder into your profile and restart<sup>1)</sup> Firefox.
5. Customise everything to your liking. (*optional*)
